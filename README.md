# Project Alliance

An AI-Nurturing Content Education platform built as a single-page web application.

## Features
- Student login system
- Interactive quizzes for EAPP, Practical Research, and Reading & Writing
- Progress tracking and scoring
- Leaderboard
- QR code generation for sharing
- Offline download capability

## Live Demo
Visit the live site at: https://rikkarikkapoop-gif.github.io/Aliance09/

## Setup
To run locally:
1. Clone the repository
2. Open `index.html` in a web browser

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
